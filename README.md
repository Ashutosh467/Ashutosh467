## 🚀 Projects

### 🗣️ CivicSense — AI Civic Complaint System
Citizens report civic issues via **phone call in any Indian language**. An AI pipeline processes, classifies, and routes complaints to the right government officer in **under 15 seconds**.

🌐 **Live:** [civicsense-alpha.vercel.app](https://civicsense-alpha.vercel.app)  
`Twilio` `Groq Whisper` `Gemini 2.0` `React.js` `Node.js` `MongoDB`

---

### 🏠 HostelHive — Hostel Management Platform
Full-stack hostel management system with an **AI chatbot that handles 80% of student queries** automatically — reducing staff load significantly.

`React.js` `Node.js` `MongoDB` `Express.js` `Firebase`

---

### 🎯 Parshikshan — Internship Recommendation Engine
Matches student profiles to **500+ internship listings** using a custom fit-score algorithm for smarter, personalized results.

`React.js` `Node.js` `MongoDB`
